About

PhD candidate at ECNU. Previously worked at Ant Group (2020-2024). Graduated from the University of Edinburgh in 2018, supervised by Dr. Myrto Arapinis.

My main interests are secure computation (MPC, HE, ZKP, etc.), and foundations of cryptography. Btw, I also use GNU Emacs. Check out my personal page at: https://jamie-cui.github.io/.

I maintain an awesome list for secure computation papers: github.com/Jamie-Cui/awesome-secure-computation

I built serveral tools for fun:

  • overleaf-project : Project-level Overleaf integration for Emacs
  • magent : An Emacs Lisp AI coding agent with multi-agent architecture, permission-based tool access, and LLM integration via gptel.
  • bilive : A lightweight, cross-platform Bilibili Live management service with a local web admin UI.
  • yacl-r : Yet Another Cryptographic Library for Research

Repos for my workflow:

  • opt : "Optional Applications". A personal collection of essential development tools and configurations managed via Makefile and TOML config.
  • dotfiles : Personal configuration for Linux and macOS desktops.
  • emacs.d : A personal Emacs configuration with two-level deferred loading, Evil mode, and gptel LLM integration. Targets Emacs 30.1+.
  • skills : My angent skills collection.

It is possible to build a cabin with no foundations, but not a lasting building.
